## Scoring

- [🟡] **Dead code in clean_preferred_languages** `suddenly/users/forms.py:25` Django's JSONField always returns a parsed list (never a `str`) in `cleaned_data` after its own `to_python`; the `isinstance(value, str)` branch is unreachable (replace with simple `return value or []`)
- [🟡] **Silent empty string on null AP URLs** `suddenly/users/models.py:75,82,89` `actor_url`, `actor_inbox`, `actor_outbox` return `""` when remote URL is null — violates Fail Fast; callers get a falsy string with no indication something is wrong
- [🟡] **ProfileEditView ignores username URL kwarg** `suddenly/users/views.py:32` `get_object` returns `request.user` regardless of the `<username>` in the URL — navigating to `/@alice/edit/` while logged in as Bob silently edits Bob's profile, not Alice's
- [🟢] **type: ignore on get_object** `suddenly/users/views.py:33` Acceptable mypy workaround given LoginRequiredMixin guarantees an authenticated User; `cast(User, self.request.user)` would be cleaner but not required
- [🟢] **add_fieldsets missing in UserAdmin** `suddenly/users/admin.py` BaseUserAdmin's add_fieldsets doesn't include new fields — new users created via admin won't see content_language etc.; low impact, admin-only
- [🟢] **content_language default 'fr' is a magic string** `suddenly/users/models.py:40` Acceptable given the French-first instance scope; could become a settings constant if multi-locale instances are planned
